كتابة الشرط

كما تذكر قبل قليل قمنا بتحديد جميع عناصر الـ radio واسندناها الى متغير بإسم options كالتالي:

var options = document.querySelectorAll("input[type=radio]");

لهذا اذا نريد أن نتحقق من اذا ما كان قد تم تحديد الجواب الصحيح والذي هو في خانة 0 كما موضح بالصورة ادناه سنقوم بتحديد العنصر الذي بتلك الخانة كالتالي options[0] وحتى نتأكد اذا ما تم عمل check عليه سنضيف الكلمة التالية .checked ومن ثم نتأكد اذا كانت خاصية checked تساوي true

الكود كالتالي:

options[0].checked == true

لكن هل تذكر أنني قمت بعمل متغير لتخزين رقم الإجابة الصحيحة 0 فيه واسميته correctAnswer لذا سأقوم بوضعه بدلاً من وضع الرقم مباشرة

الكود الأن كالتالي:

options[correctAnswer].checked == true

اذا سيكون كود الشرط كالتالي:

if (options[correctAnswer].checked == true) {

	// قم بإضافة التنسيق الخاص بالإجابة الصحيحة للعنصر
	// قم بفتح النافذة المنبثقة الخاصة بالجواب الصحيح

} else {

	// قم بفتح النافذة المنبثقة الخاصة بالجواب الخطأ

}

لنقم الأن بإضافة التنسيق المطلوب للإجابة الصحيحة في حال تم إختيارها

هل تذكر سابقاً قمنا بعمل الكلاس correct واضفنا الية التنسيقات التي نرغب بتطبيقها على الجواب الصحيح اذا ما تم إختيارة لهذا نريد ان نقوم بكتابة كود يقوم بإضافة هذا الكلاس للعنصر li الخاص بالإجابة الصحيحة التي تم تخزين رقمها في المتغير correctAnswer

سنقوم بتحديد العنصر الخاص بالإجابة الصحيحة كالتالي li[correctAnswer] ومن ثم سأتبعة بـ .classList والتي ستحدد قائمة الكلاسات التي لدى هذا العنصر ومن ثم سأطلب إضافة كلاس جديد .add بإسمcorrect

الكود كالتالي:

li[correctAnswer].classList.add("correct");

البدء بالعمل

قمنا سأبقاً بتجهيز النافذة المنبثقة ومن ثم قمنا بإخفائها بإعطائها الخاصية display:none حتى نقوم بإظهارها لاحقاً عند حاجتنا لها, لهذا سنقوم الأن بتغيير خاصية العرض display من none الى block حتى تظهر النافذة وللقيام بذلك:

اولاً سنحتاج لتحديد العنصر الخاص بهذه النافذة حتى نستطيع الوصول الية عبر الجافاسكربت والتلاعب به, كما قمنا سابقاً سأقوم بتعريف متغير بإسم modal وسأسند الية تحديد العنصر الذي يحمل id بإسم overlay حيث وانه الـ id الخاص بهذه النافذة قم بمراجعة اكواد النافذة.

var modal = document.getElementById("overlay");

ثانياً بعد تحديد العنصر نستطيع الان التلاعب به وما نريد القيام به هو تغيير تنسيق display وحتى نقوم بذلك سنقوم اولاً بتحديد العنصر الذي نريد التلاعب به وهو modal ومن ثم سأحدد بأنني اريد التلاعب بالتنسيق الخاص به بإستخدام .style بعد ذلك سأحدد الخاصية التي اريد تعديلها وهي .display واخيراً سأعطيها القيمة الجديدة والتي هي block كالتالي:

modal.style.display = "block";

بنفس الفكرة سأقوم بإضافة صورة الإجابة الخاطئة كالتالي:

العمل على الملاحظة الثانية:

عند فتح النافذة المنبثقة لا نستطيع إغلاقها الا بتحديث الصفحة فما اريد القيام به الان هو جعل هذه النافذة تغلق عند الضغط على اي مكان في نافذة المتصفح خارج النافذة المنبثقة على منطقة overlay تحديداً حيث الخلفية السوداء الشفافة:

كما قمنا سابقاً مع العناصر التي قمنا بالإنتظار للضغط عليها لتنفيذ بعض الاحداث هذه المرة سننتظر ان يتم الضغط على نافذة المتصفح كالتالي:

window.addEventListener('click', function () {

	// يتم تنفيذ هذه الاوامر عند الضغط في أي مكان بالصفحة

});

كما تلاحظ الكود اعلاة يقوم بتنفيذ الاوامر عند الضغط في اي مكان على الصفحة ونحن لا نريد تنفيذ الاوامر الا عندما يكون العنصر المستهدف بهذا الحدث event.target هو العنصر modal

لذا سأضع الشرط التالي:

window.addEventListener('click', function () {

	if (event.target == modal) {

		// يتم تنفيذ هذه الاوامر عند الضغط على العنصر المحدد فقط

	}

});

اذا تم الضغط على منطقة العنصر modal قم بإعطاء هذا العنصر خاصية display:none لإخفائة.

window.addEventListener('click', function () {

	if (event.target == modal) {

		modal.style.display = "none";

	}

});

تذكر قمنا بتحديد العنصر overlay بإستخدام المتغير modal لهذا فتعاملك مع المتغير modal في الجافاسكربت هو نفس تعاملك مع العنصر overlay في الـHTML و الـ CSS.

هذا ما يسمى بالمتغير أو الـ variable في الـ Javascript حيث يتيح لك تخزين قيمة فيه لإستخدامها عند الحاجة لها وكذلك يسمح لك بتغييرها لاحقاً

كيفية تعريف الثابت (constant) في الـ Javascript:

لتعريف ثابت يجب أن تعطية إسم وتسبق هذا الإسم بـ const والتي هي إختصار من constant ومن ثم تقوم بإعطائة قيمة بنفس السطر كالتالي:

const number = 5;

ما قمت به في الكود أعلاة هو تعريف ثابت (constant) بإسم number وقمت بتخزين الرقم 5 بدأخلة.

كيفية تعريف المتغير (variable) في الـ Javascript:

الطريقة الأولى:

لتعريف متغير يجب أن تعطية إسم وتسبق هذا الإسم بـ var والتي هي إختصار من variable ومن ثم تقوم بإعطاء المتغير قيمة بنفس السطر كالتالي:

var result= 0;

ما قمت به في الكود أعلاة هو تعريف متغير (variable) بإسم result واسندت الية قيمة إبتدأية تساوي 0

الطريقة الثانية:

لتعريف متغير يجب أن تعطية إسم وتسبق هذا الإسم بـ var ثم تتركة بدون أي قيمة وفي وقت لاحق تقوم بإسناد قيمة لهذا المتغيرعند حاجتك له كالتالي:

var result;

في الكود أعلاة قمت بتعريف متغير ولم أسند إلية أي قيمة ولنفترض أنني إحتجت لإستخدامه بعد الف سطر من الأكواد البرمجية حينها سأقوم بإعطائة القيمة في ذلك المكان بإستخدام الكود التالي:

result = 0;

لاحظ عندما أقوم بإسناد قيمة إلية لاحقاً أقوم بكتابة إسم المتغير فقط وإسند إلية القيمة مباشرة بعد علامة يساوي =

بعد أن عرفنا الفرق بينهما لنقوم بعمل كود اللعبة أعلاة

سنفترض بإنك إخترت الرقم 5 سيكون كود الـ Javascript كالتالي:

<script>
    const number = 5; // إختر رقم في عقلك و لا تقوم بتغييرة

    var result  = number * 2;  // اضرب الرقم الذي في عقلك بـ 2

    result = result + 8; // أضف للناتج الرقم 8

    result = result / 2; // إقسم الناتج على 2

    result = result - number; // إطرح من الناتج أول رقم وضعته في عقلك

    console.log(result); // هذا الكود سيقوم بإظهار الناتج في نافذة منبثقة

 </script>
  • قمنا بتعريف الرقم كثابت const وأسندنا إلية الرقم 5
const number = 5;
  • قمنا بتعريف المتغير result وأسندنا إلية القيمة 10 من خلال العملية الحسابية number*2 حيث أن number سيتم التعويض عنه بـ 5
var result  = number * 2;
  • قمنا بتحديث قيمة المتغير result إلى 18 من خلال العملية الحسابية result+8 حيث سيتم التعويض عن result بالقيمة السابقة والتي هي 10 ليصبح لدينا result=10+8, ولاحظ لم نكتب قبلة var وذلك لانه لا يتطلب ذلك الا عند تعريفة أول مرة
result = result + 8;
  • قمنا بتحديث قيمة المتغير result إلى 9 من خلال العملية الحسابية result/2 ما يقابلها 18/2:
result = result / 2;
  • قمنا بتحديث قيمة المتغير result إلى 4 من خلال العملية الحسابية result-number ما يقابلها 9-5 بعد التعويض بقيم كلاً من الثابت number و المتغير result في المعادلة.
result = result - number;
  • قمنا بطباعة النتيجة بإستخدام امر alert الذي يقوم بفتح نافذة منبثقة
alert(result);
